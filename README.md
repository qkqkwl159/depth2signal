# depth2signal

**First my repo clone**

```git
git clone https://github.com/qkqkwl159/depth2signal.git
```

**Move dir**

```shell
cd depth2signal
```

**Clone MiDaS**

```git
git clone https://github.com/isl-org/MiDaS.git
```

**conda excute on base**

```shell
conda env create -f environment.yml
```

```shell
conda activate c2f
```

**Run**
```shell
python3 cam2frame.py
```
